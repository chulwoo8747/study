# github project로 project 관리하기

## 그렇다면 Project란?

프로젝트는 작업 현황과 진행도를 볼 수 있는 메뉴이다. 이슈, PR(풀 리퀘스트)들을 하나의 작업으로 구분해 그 작업이 현재 어느 정도 진행되었는지 확인할 수 있다. 

- To Do : 해야 할 작업
- In Progress: 진행 중인 작업
- Done : 완료된 작업

이런식으로 Projects를 사용하면 위와 가타은 기능들을 제공해주는데 이것을 통해 프로젝트의 진행상황을 한번에 파악하기가 수월다는 장점이 있다. 또한 프로젝트에서 일어나는 모든 일을 추적할 수 있고, 마지막으로 본 이후에 어떤 일이 생겼는지 확인할 수 있다.
