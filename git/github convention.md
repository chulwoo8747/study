# github convention

### Commit convention

```bash
[commit type] 본인이 작업 하는 것

예시
[FEAT] 로그인 기능 구현 완료 // commit type은 대문자로 구성
```

**commit type**

| TYPE   | 의미                   |
| ------ | -------------------- |
| FEAT   | 새로운 기능 개발 및 파일 생성    |
| STYLE  | 스타일 관련 코드, css 파일 생성 |
| FIX    | 오류 고침, 파일 구조 변경      |
| DELETE | 파일 삭제, 코드 삭제         |

### Branch convention

```bash
FEAT/(issue number)-개발 내용
```
